# Love Token Website Expansion Plan

## Overview

This document provides a comprehensive roadmap for expanding the Love Token website to fully communicate the vision, mechanics, and impact of the Love Economy. It references the core project documentation (`Love App Overview - Dan Lawless.docx.md`) and outlines specific content additions, infographics, and narrative enhancements for each page.

---

## Guiding Principles (from Project Plan)

Before expanding content, all additions should align with these core principles:

1. **Dignity First** - Communities participate as equals, never as recipients
2. **Love as Currency** - Real, practical, measurable value creation
3. **Simplicity is Respect** - Design for universal understanding
4. **Quality Over Speed** - Correctness over polish, trust over speed
5. **Philosophy**: "Love is not given. Love is shared. Love is earned. Love circulates."

## Content Depth Guidelines

### Writing Style
- Clear, simple language
- Active voice
- Dignity-first framing
- Never "beneficiaries" — always "participants" or "Love Farmers"
- Economic language (value, currency, investment) not charity language

### Content Hierarchy
1. **Hero Statement**: One powerful sentence
2. **Subhead**: Context in 1-2 sentences
3. **Body**: Detailed explanation (3-5 paragraphs max)
4. **Visual**: Infographic or image
5. **CTA**: Clear next action

### Accessibility Considerations
- All infographics have text alternatives
- Symbol-based UI for key interactions
- High contrast text
- Large touch targets
- Screen reader compatible

## Success Metrics

The website expansion succeeds when:

1. **Clarity**: A first-time visitor understands the Love Economy in under 2 minutes
2. **Depth**: Each role (Farmer, Donor, Vendor) has a complete understanding path
3. **Trust**: Security and governance concerns are proactively addressed
4. **Inspiration**: The vision feels achievable, not utopian
5. **Action**: Clear CTAs at every stage of the journey
